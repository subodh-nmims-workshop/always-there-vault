## 📱 Professional React Native Application

### Features Implemented

#### 🔐 **Security Services**
- **BiometricService**: Fingerprint/Face ID authentication
- **CryptoService**: AES-256-GCM encryption + Shamir Secret Sharing
- **StorageService**: Secure local storage with encryption

#### 🎨 **Professional UI Components**
- **Card**: Reusable card component with elevation
- **Button**: Multi-variant button (primary, secondary, outline, ghost, danger)
- **Input**: Professional text input with validation and icons

#### 📱 **Complete Screens**
- **HomeScreen**: Dashboard with stats and quick actions
- **AssetsScreen**: Asset management with encryption
- **BeneficiariesScreen**: Beneficiary management
- **HeartbeatScreen**: Animated proof-of-life monitor
- **SettingsScreen**: Comprehensive settings management

#### 🛠 **Utilities & Types**

- **Validators**: Comprehensive validation functions
- **Types**: Complete TypeScript type definitions

### Installation & Setup

```bash
# Install dependencies
npm install

# iOS Setup (if on macOS)
cd ios && pod install && cd ..

# Run on Android
npm run android

# Run on iOS
npm run ios

# Start Metro bundler
npm start
```
